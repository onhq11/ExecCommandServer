## Requirements
- NodeJS preinstalled

## Installation
- Clone repository
- Enter to folder and execute ```npm i```
- Run server using ```node index.js```

## URL Requests
- Export PIN ```/export<PIN>```
- Unexport PIN ```/unexport<PIN>```
- Set PIN direction to out ```/out<PIN>```
- Set GPIO state ```/gpio<PIN>-<1/0>```
- Run TV Script ```/tv_<on/off>```
